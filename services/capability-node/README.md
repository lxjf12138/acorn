# Capability Node

A deployable runtime node that hosts Capability Providers.

Responsibilities:

- provider manager
- MCP Agent Surface
- signal emitting
- node control APIs
- observation events
- local governance hooks
- workspace and resource management
- provider runtime

Deployment modes:

- personal local node
- cloud node
- enterprise intranet node
- edge node

Non-goals:

- does not own global agent session transcript
- does not perform model reasoning by default
- does not decide global trigger rules
